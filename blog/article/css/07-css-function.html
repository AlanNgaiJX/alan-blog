# counters

# 使用过程：

  1. 使用counter-reset: item; 初始化变量名item，并重置计数器为0
  2. couter-increment: item; 计数器+1
  3. content: counter(item); 展示item现在的值
body {
  counter-reset: section;            
}

h3::before {
  counter-increment: section;        
  content: "Section " counter(section) ": ";                             
}
